## 1️⃣ How does it work?

NoMoreAuto uses EasyOCR to convert screenshots of your player leaderboard into Roblox user names, comparing them against our database of 95,000+ VERIFIED autobuilders/exploiters. We do not guarantee alt detection or 100% accuracy.


<br>

----


## 2️⃣ Getting started

- Install the exe:
  
- Go to -> AutoBuilder Scan
- Press Esc in roblox to access the leaderboard
- Press -> snip an area
- Select the area with Roblox usernames

<br>

----


## 3️⃣ Extra info & legal.

NoMoreAuto does NOT break the Roblox TOS as it simply takes a picture of the leaderboard overlay.

It is fully open-source, and none of your data is ever stored.
